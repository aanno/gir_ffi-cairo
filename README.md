# GirFFI-Cairo

GirFFI-based bindings for Cairo

## Usage

    require 'gir_ffi-cairo'

    dst = Cairo::ImageSurface.new(:argb32, 400, 300)
    ctx = Cairo::Context.new(dst)

## Install

    gem install gir_ffi-cairo
